# Drizzle Migrations

Este directório é gerido automaticamente pelo Drizzle Kit.

Para aplicar o schema à base de dados Neon:

```bash
npm run db:push
```

Para gerar ficheiros de migração SQL:

```bash
npx drizzle-kit generate
```

Para aplicar migrações:

```bash
npx drizzle-kit migrate
```

Para abrir o Drizzle Studio (GUI):

```bash
npm run db:studio
```
